### データベース関連

- csv
  - テストデータを csv 形式で保存
  - outputCSV.py で生成する
  - company.csv
  - user.csv
  - companyInfo.csv
- json
  - テストデータを json 形式で保存
  - company.json
  - user.json
  - companyInfo.json
- dbReset.py
  - データベースを json フォルダの内容にリセットする
- outputCSV.py
  - json フォルダの内容を csv フォルダに形式を変換して保存する

### データ設計

#### user

ユーザのスキーマ

- email:Email :主キー
- name:ユーザー名
- password:パスワード
- companyInfoList companyInfo のリスト

#### companyInfo

企業とそのメモのスキーマ

- email: Email 複合キー
- id:企業 ID 複合キー
- memo:メモ

#### company

企業のスキーマ

- id:ID
- name:企業名
- abstract:概要
- industries:業種のリスト
- locations:勤務地のリスト
